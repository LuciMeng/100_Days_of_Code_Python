# Day 19 - Turtle Racing 

## Project Overview
Built a Turtle Racing Game using Python’s Turtle graphics module. The user places a bet on which colored turtle will win, and the turtles race across the screen at random speeds until a winner is declared.

## What I Learned
- Creating and managing multiple turtle objects (instances)
- Using `screen.textinput()` for user input
- Working with event listeners and higher-order functions
- Using lists to store and control multiple objects
- Reading Turtle documentation to find and apply methods

## How It Works
- Create a race screen and ask the user to place a bet
- Generate multiple turtles with different colors and positions
- Store turtle objects in a list for movement control
- Move each turtle forward randomly inside a race loop
- Detect the first turtle to cross the finish line
- Display whether the user’s bet was correct

## Key Takeaways
- Improved understanding of object instances and object management
- Practiced loops, random movement, and coordinate checking
- Learned how to build interactive programs with Turtle graphics
