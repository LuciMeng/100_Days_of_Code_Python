# Day 18 - Hirst Painting Project

## Project Overview
Created a digital Damien Hirst-inspired dot painting using Python and the Turtle graphics module. The program generates a 10×10 grid of randomly colored dots using RGB color tuples extracted from an image.

## What I Learned
- Importing and using modules such as `turtle` and `colorgram`
- Extracting RGB colors from images with `colorgram`
- Working with lists and tuples for color storage
- Using Turtle methods like `.dot()`, `.penup()`, and `colormode(255)`
- Reading Python documentation to understand module methods

## How It Works
- Extract RGB colors from an image and store them as tuples
- Set up the turtle with fast speed and hidden pen
- Use nested loops to draw a 10×10 grid of colored dots
- Randomly select a color for each dot

## Key Takeaways
- Practiced using external Python modules
- Improved understanding of loops and coordinate movement
- Learned how RGB coloring works in Turtle graphics
